# Claude Plugins

A collection of [Claude Code](https://claude.ai/code) plugins.

## Plugins

| Plugin | Description |
|--------|-------------|
| [smith](smith/) | Research, Plan, Implement Pipeline |
| [otel](otel/) | Tools for OpenTelemetry |

## Installation

Add the marketplace, then install any plugin:

```
/plugin marketplace add https://github.com/brettbuddin/claude-plugins
/plugin install smith@brettbuddin
```
