# OTel

Tools for working with [OpenTelemetry](https://opentelemetry.io/).

## Install

Add the marketplace and install:

```
/plugin marketplace add https://github.com/brettbuddin/claude-plugins
/plugin install otel@brettbuddin
```

## Commands

| Command | Description |
|---------|-------------|
| `/otel:collector-component` | Scaffold a new OpenTelemetry Collector component (receiver, processor, exporter, connector, or extension) |
